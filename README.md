# admin_group_proto

동아리 관리 시스템 프로토타입

## 프로젝트 개요

동아리 관리 및 커뮤니티 플랫폼 프로토타입입니다.

## 기술 스택

- React 19.2.0
- TypeScript 4.9.5
- Supabase
- Create React App

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```

개발 서버는 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 빌드

```bash
npm run build
```

프로덕션 빌드가 `build` 폴더에 생성됩니다.
